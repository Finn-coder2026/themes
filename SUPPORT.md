# Getting help

## Questions about themes

Start a Discussion if you're unsure about theme usage or want to share ideas:

- <https://github.com/jolliai/themes/discussions>

## Bug reports

Open an issue using the **Bug report** template:

- <https://github.com/jolliai/themes/issues/new/choose>

Please include:

- Which theme is affected and its version
- The Jolli CLI version (`jolli --version`)
- Steps to reproduce
- Expected vs actual behavior

## Feature requests

Open an issue using the **Feature request** template. The more concretely you can describe the design you'd like to see, the easier it is for us to scope.

## Jolli CLI issues

For problems with the CLI itself (not theme-specific), report at the main repository:

- <https://github.com/jolliai/jolliai/issues/new/choose>

## Security vulnerabilities

**Please do not file public issues for security problems.** See [`SECURITY.md`](SECURITY.md) for the private reporting process.
